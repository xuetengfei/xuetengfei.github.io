### 1.Install on MacOS

```shell
brew tap mongodb/brew
brew install mongodb-community@4.2
```

### 2.start service with brew

```shell
brew services start mongodb-community@4.2
```

### 3.Use MongoDB Shell

```
$ mongo
```
